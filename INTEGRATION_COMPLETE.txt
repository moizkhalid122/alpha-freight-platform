===========================================
HOW TO USE IN OTHER PAGES
===========================================

Step 1: Add Scripts (before </body>)
------------------------------------
<script src="../assets/js/error-handler.js"></script>
<script src="../assets/js/toast-notifications.js"></script>
<script src="../assets/js/skeleton-loader.js"></script>
<script src="../assets/js/lazy-load.js"></script>
<script src="../assets/js/form-validator.js"></script>

Step 2: Replace showAlert with Toast
------------------------------------
// OLD:
showAlert('Success!', 'success');

// NEW:
showSuccess('Success!');

Step 3: Add Skeleton Loader
----------------------------
// Show skeleton while loading
SkeletonLoader.show('containerId', 'load-card', 3);

// Hide when data loaded
SkeletonLoader.hide('containerId');

Step 4: Add Form Validation
----------------------------
// In HTML:
<input type="email" name="email" data-validate>

// In JavaScript:
const validator = new FormValidator('formId');
validator.addRule('email', 'required', 'Email required');
validator.addRule('email', 'email', 'Invalid email');

Step 5: Use Lazy Loading
------------------------
// For images:
<img data-src="image.jpg" alt="Description">

// For content:
<div data-lazy="/path/to/content.html">Loading...</div>

===========================================
QUICK REFERENCE
===========================================

Toast Notifications:
- showSuccess('Message')
- showError('Message')
- showWarning('Message')
- showInfo('Message')

Skeleton Loader:
- SkeletonLoader.show(id, type, count)
- SkeletonLoader.hide(id)

Form Validator:
- new FormValidator('formId')
- validator.addRule(field, rule, message)

Error Handler:
- Automatic (catches all errors)
- handleError(error, message)

Lazy Loading:
- Automatic (just use data-src instead of src)
